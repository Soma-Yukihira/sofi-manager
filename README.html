⚜ Selfbot Manager

Application graphique pour gérer plusieurs selfbots Discord SOFI en parallèle. Interface premium black & gold, thème clair, couleurs personnalisables, logs en direct par bot.

✨ Fonctionnalités

  • Plusieurs selfbots gérés depuis une seule interface (un thread par bot)
  • Tri automatique des wishlists (personnages et séries) à la sauvegarde
  • Détection multilingue des drops SOFI (FR / EN)
  • Pause nocturne aléatoire entre 22h et 01h
  • Système de scoring rareté + popularité avec override wishlist
  • Thèmes : preset sombre, preset clair, couleurs entièrement personnalisables
  • Logs colorés en temps réel par bot

📋 Prérequis

  • Python 3.10+ (testé sur 3.14)
  • pip disponible dans le terminal

Vérifier :

python --version
pip --version

🚀 Installation

1. Récupérer le projet

Décompresser l'archive dans un dossier de ton choix, par exemple :

C:\Users\TonNom\Documents\SELFBOT

2. Créer un environnement virtuel (recommandé)

Ouvrir un terminal dans le dossier du projet :

python -m venv env
.\env\Scripts\activate
Sous macOS / Linux : source env/bin/activate

3. Installer les dépendances

pip install -r requirements.txt

Cela installe :

  • discord.py-self — client Discord (selfbot)
  • customtkinter — moteur graphique premium

4. Lancer l'application

python main.py

L'interface s'ouvre. Aucune config n'est partagée — bots.json est créé automatiquement à ton premier ajout de bot.

🎮 Utilisation

Ajouter un bot

  1. Cliquer sur + AJOUTER UN BOT (sidebar gauche)
  2. Onglet Configuration :
    • Nom : libre, pour t'y retrouver
    • Token Discord : ton token (masqué)
    • ID du salon de drop : où envoyer la commande sd
    • Salons écoutés : un ID par ligne — le drop channel est inclus automatiquement
  3. Onglet Wishlist : tes personnages favoris et séries favorites, un par ligne (le tri se fait à la sauvegarde)
  4. Cliquer Sauvegarder, puis ▶ Démarrer

L'onglet Logs s'ouvre automatiquement et tu vois en direct ce que fait ton bot. Le statut du bot apparaît dans la sidebar (point vert = en marche).

Personnaliser le thème

  • ☀ Clair / 🌙 Sombre : bascule entre les deux presets
  • 🎨 Couleurs : ouvre la palette pour modifier chaque couleur individuellement (fond, accent, succès, erreur, etc.). Cliquer sur une couleur ouvre un sélecteur. Réinitialiser revient au preset courant.

Les préférences de thème sont stockées dans settings.json.

Gérer plusieurs bots

Chaque bot a son propre thread, sa propre boucle asyncio et ses propres logs. On peut les démarrer/arrêter indépendamment et basculer d'un bot à l'autre via la sidebar.

⚠️ Sécurité

  • Les tokens sont stockés en clair dans bots.json — ne partage jamais ce fichier ni ne le commit sur Git.
  • Ne montre jamais ton token à quelqu'un, même un ami : il aurait un accès complet à ton compte Discord.
  • Les selfbots sont contre les ToS Discord. À tes risques.

🐛 Dépannage

« Token invalide »

Le token a expiré ou est mal copié. Re-copie-le depuis Discord (DevTools → Network).

Le bot ne détecte pas les drops

Regarde les lignes 📥 SOFI: dans les logs : elles affichent ce que SOFI envoie. Si le format a changé, signale-le.

ModuleNotFoundError: No module named 'discord'

L'environnement virtuel n'est pas activé ou les dépendances ne sont pas installées. Refais l'étape 2 + 3.

ModuleNotFoundError: No module named 'customtkinter'

Idem — pip install -r requirements.txt dans le venv activé.
